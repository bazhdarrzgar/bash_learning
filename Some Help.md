## Change Shell


```bash

echo $SHELL

---
/bin/sh
---

# if you want change it to bash use this


chsh -s /bin/bash 

echo $SHELL

---
/bin/bash
---


# if you want use sh or bash just type it

sh 

---
# you are ready to type something
$
# exit if you want exit in this terminal
---


# run command with bash or sh

bash hello.sh

sh hello.sh

```
